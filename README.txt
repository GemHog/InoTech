InoTech — Наше антивирусное программное обеспечение осуществляет полное сканирование системы, чтобы обнаружить и удалить вирусы, трояны, черви и другие угрозы.

Инструкция по запуску:
1. Загрузите все файлы проекта, такие как HTML, CSS, JS и изображения, и разместите их в папке на вашем веб-сервере.
2. Запустите файл index.html в любом удобном браузере.
3. Проведите проверку всех функций вашего сайта, чтобы удостовериться, что все работает корректно.
